## git clone 
到你想要创建项目的目录下，执行

```shell
git clone https://github.com/rocwong-cn/teach-girl-friend-git.git
```
会生成项目目录

## git add 

```shell
git add <file>
```
把文件加到Git本地暂存区


## git commit

```shell
git commit -a -m '说明文字'
```

## git push
推送本地暂存区的代码到远端服务器；

## git stash / git stash pop
代码临时缓存
